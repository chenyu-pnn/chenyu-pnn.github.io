# 3D Models

Drop GLB/GLTF files here. The viewer will auto-load them if the filename matches the project `id` in `data.js`.

| File | Project |
|---|---|
| `neural-mesh.glb` | Neural Mesh Renderer |
| `quantum-sim.glb` | Quantum Circuit Simulator |
| `tactile-net.glb` | Tactile Sensor Network |

If a file is missing the viewer falls back to the procedural geometry automatically.
